# Java POO

Bienvenido a mi repositorio de especialización en **Java**. Este espacio documenta mi recorrido académico y técnico en el aprendizaje del lenguaje, desde la lógica algorítmica básica hasta la implementación de arquitecturas complejas basadas en **Programación Orientada a Objetos (POO)**.

---
